# Plan 03-03: Recipes Frontend — Weekly Meal Planner

## Goal
A weekly view where each day has 4 meal slots (Breakfast, Lunch, Dinner, Snack). Family members can assign a recipe to any slot by picking from the recipe list.

## Tech Stack Reminders
- Next.js 16.2.1, React 19, Tailwind CSS v4
- `useActionState` for mutations
- `apiFetch` from `frontend/lib/api.ts`

## Scope
- Weekly planner view (Mon–Sun)
- Click on empty slot → recipe picker modal
- Click on filled slot → show recipe name, click remove to clear
- Week navigation (prev/next week buttons)
- Today highlighted

## Out of Scope
- Recipe CRUD (03-02)
- Shopping list generation from meal plan
- Multi-week view

## Implementation

### 1. Add to `frontend/lib/recipes.ts`
Types: `MealPlan`, `MealSlot` (BREAKFAST | LUNCH | DINNER | SNACK)
Functions:
- `getMealPlan(from: string, to: string): Promise<MealPlan[]>`
- `setMealPlan(date: string, slot: MealSlot, recipeId: string, notes?: string): Promise<MealPlan>`
- `removeMealPlan(date: string, slot: MealSlot): Promise<void>`

### 2. `frontend/components/recipes/RecipePicker.tsx`
- Modal overlay
- Search input with live filter
- Scrollable list of recipes
- Click recipe → calls onSelect(recipe)
- ESC or backdrop click → closes

### 3. `frontend/components/recipes/MealSlotCell.tsx`
- Empty: shows "+" button with slot label
- Filled: shows recipe name + small "×" remove button
- Loading spinner while mutation in flight
- Optimistic update

### 4. `frontend/components/recipes/WeeklyPlanner.tsx`
- Grid: rows = 4 slots, columns = 7 days
- Header row: day names + dates (Mon 6 Jan style)
- Today's column has subtle highlight
- Uses MealSlotCell for each cell
- Manages open/close state for RecipePicker modal

### 5. `frontend/app/(app)/recipes/meal-plan/page.tsx`
- Week state: ISO week start (Monday)
- Prev/Next week buttons + "This week" jump button
- Fetches meal plan for current week
- Renders WeeklyPlanner
