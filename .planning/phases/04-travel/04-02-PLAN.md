# Plan 04-02: Travel Frontend — Trip Overview + Packing Lists

## Goal
Trip list, trip creation, and packing lists (shared + personal). Mobile-friendly — packing lists especially need to work well on phones.

## Tech Stack Reminders
- Next.js 16.2.1, React 19, Tailwind CSS v4
- `useActionState` for forms
- `apiFetch` from lib/api.ts
- No react-hook-form, no localStorage

## Scope
- `frontend/lib/travel.ts` — typed API functions for trips and packing
- Trip list page with upcoming/past sections
- Trip detail page (summary, key info panel, tabs for packing/itinerary)
- Create trip form
- Shared packing list with optimistic toggle
- Personal packing list (my items only) with toggle
- Add packing item inline (text input + add button)

## Out of Scope
- Itinerary (04-03)
- Cover photo upload (nice to have, skip for now)
- Edit trip form (can add later)

## Implementation

### 1. `frontend/lib/travel.ts`
Types: `Trip`, `PackingItem`, `ItineraryEntry`, `TripKeyInfo`, `TripRequest`, `PackingItemRequest`
Functions:
- `getTrips(): Promise<Trip[]>`
- `getTrip(id: string): Promise<Trip>`
- `createTrip(data: TripRequest): Promise<Trip>`
- `deleteTrip(id: string): Promise<void>`
- `getPackingItems(tripId: string, personal?: boolean): Promise<PackingItem[]>`
- `addPackingItem(tripId: string, data: PackingItemRequest): Promise<PackingItem>`
- `togglePackingItem(tripId: string, itemId: string): Promise<PackingItem>`
- `deletePackingItem(tripId: string, itemId: string): Promise<void>`

### 2. `frontend/components/travel/TripCard.tsx`
- Cover photo (with fallback: gradient based on destination initial)
- Destination + date range (e.g. "23 Jul – 3 Aug 2025")
- Days until trip / "In progress" / "X days ago"
- Link to detail page

### 3. `frontend/components/travel/PackingList.tsx`
- Props: tripId, personal (boolean), currentUserId
- Loads items on mount
- Each item: checkbox (optimistic toggle), label, delete button
- "Add item" inline: text input at bottom + Enter/button to submit
- Strikethrough on packed items
- Shows progress: "5 / 12 packed"

### 4. `frontend/components/travel/TripForm.tsx`
- Fields: title, destination, startDate, endDate, description
- `useActionState`
- Date inputs: type="date"
- Redirects to /travel/{id} on success

### 5. `frontend/app/(app)/travel/page.tsx`
- Two sections: Upcoming trips (start_date >= today) and Past trips
- Grid of TripCards
- "Plan a Trip" button → /travel/new
- Empty state per section

### 6. `frontend/app/(app)/travel/new/page.tsx`
- Wraps TripForm

### 7. `frontend/app/(app)/travel/[id]/page.tsx`
- Header: destination, dates, description
- Key Info panel: label/value pairs (from trip.keyInfos)
- Two tabs: "Packing List" and "Itinerary" (itinerary tab just shows "coming soon" placeholder)
- Under Packing List tab:
  - Sub-tabs: "Shared" | "My Items"
  - Renders PackingList component with appropriate props
- Delete trip button (creator/admin only) with confirm

2. Create trip form works and redirects to detail
3. Detail page shows key info and packing list tabs
